# IKA Car (website-car)

TanStack Start application (Vite, React 19, TanStack Router).

## Development

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
bun run build
bun run preview
```

See `implementation_summary.md` for migration notes and architecture details.
